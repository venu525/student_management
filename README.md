# Student Management System

A simple Student Management System using PHP & MySQL.

## Features
- Add Student
- View Student List
- Edit Student
- Delete Student

## Technologies Used
- PHP
- MySQL
- HTML
- CSS

## Files Description
- index.php – Display students
- db.php – Database connection
- edit.php – Edit student
- delete.php – Delete student

## How to Run
1. Install XAMPP
2. Copy project to `htdocs`
3. Create database in phpMyAdmin
4. Import table
5. Run in browser: http://localhost/student_management
